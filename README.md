Distributed Machine Learning Platform.

prerequisite:

1. python 3.7
2. ```conda install -c anaconda scons```
3. Clone this repository
4. Update cppSANN submodule ```git submodule update --init --recursive```
5. Call ```./build.sh``` to build the project.
